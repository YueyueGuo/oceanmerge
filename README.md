# OceanMerge

A Suika-style physics puzzle game where you drop and merge kawaii sea creatures to release baby seals!

## Features

- 10 unique sea creatures with hand-drawn kawaii visuals
- Physics-based gameplay using Matter.js
- Merge matching creatures to evolve them
- Goal: Merge two Turtles to set the Baby Seal free!

## Tech Stack

- React 18 + TypeScript
- Matter.js (physics engine)
- Vite (build tool)
- Tailwind CSS
- Capacitor (iOS deployment)

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the app:
   ```bash
   npm run dev
   ```

## Build

```bash
npm run build
```

## iOS Deployment

Uses Capacitor for native iOS app packaging. See `ios/` directory.
